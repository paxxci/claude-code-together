# Architecture SOP: Hero Section

## Goal
Create a high-impact, editorial-style hero section that communicates the "Dying Breed" brand essence.

## Design
- Big bold headline (Serif, Playfair Display).
- Gold accent colors (#D4AF37).
- Breathing room (large padding).
- Illustration/Mockup on the right.

## Inputs
- Headline: "For Those Who Refuse to Fade Away"
- CTA: "Buy Hat Now"

## Implementation Logic
1. Flexible container using Flexbox/Grid.
2. Responsive font sizes (text-5xl at mobile, text-8xl at desktop).
3. CTA button with hover scale effects.
